# Game Analyzer

Refereeなどから直接示されない，ゲームの状況を解析するノード．  
ゲームの状況を解析し，ゲームの状況を示すメッセージを毎フレームpublishする．  
重い処理を含む解析は数フレームに1度のみ行い，それ以外のフレームでは前回の結果をpublishする．

## ボール所持判定

両チームのボールに最も近いロボットを抽出し，ボールとの距離がしきい値以下ならばボール所持と判定する．
